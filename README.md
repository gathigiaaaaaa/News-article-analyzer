# News Article Analyzer

## Description

This Python application analyzes a news article and provides several useful statistics about the text.

## Features

1. Substring Match & Specific Word Count
2. Regex Match & Identify Most Common Word
3. Calculate Average Word Length
4. Count Paragraphs
5. Count Sentences

## How to Run

Open a terminal.

Navigate to the src folder.

```
cd src
```

Run the application.

```
py main.py
```

or

```
python main.py
```
